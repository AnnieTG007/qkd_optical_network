# SpRS（自发拉曼散射）噪声建模公式

## 使用的参数（详见 parameters.md）

核心参数：
- α₁：信号光频率 f₁ 处的光纤衰减 [1/m]
- α₂：泵浦光频率 f₂ 处的光纤衰减 [1/m]
- g_R(f₁,f₂)：拉曼系数 = g_eff/A_eff [1/(W·m)]
- n_th(Δf)：声子占据因子 [无量纲]
- Δf = |f₁ - f₂|：泵浦-信号频率差 [Hz]
- h：普朗克常数 = 6.626e-34 [J·s]
- k：玻尔兹曼常数 = 1.381e-23 [J/K]
- T：工作温度 [K]

离散模型额外参数：
- P_{f,2}, P_{b,2}：前向/后向泵浦光功率 [W]

连续模型额外参数：
- G_{f,2}, G_{b,2}：前向/后向泵浦光功率谱密度 [W/Hz]

---

## 1. 声子占据因子（公式3.1.2）

```
n_th(Δf) = 1 / [exp(h×Δf / (k×T)) - 1]
```

物理说明：
- T → 0K 时，n_th → 0，反斯托克斯分量消失
- 室温(300K)下，小频移时 n_th 显著，反斯托克斯分量不可忽略
- Δf 很大时，n_th → 0

---

## 2. 拉曼横截面积 σ_{1,2}（公式3.1.4）

斯托克斯过程（f₂ > f₁，泵浦频率高于信号频率）：
```
σ_{1,2} = 2×h×f₁ × g_R(f₂,f₁) × (1 + n_th(Δf)) × Δf
```

反斯托克斯过程（f₂ < f₁，泵浦频率低于信号频率）：
```
σ_{1,2} = 2×h×f₁ × g_R(f₁,f₂) × n_th(Δf) × (f₁/f₂) × Δf
```

单位：[1/m]

注意：σ_{1,2} 中的 Δf 是泵浦-信号频率差 |f₁-f₂| [Hz]，不是积分微元。此定义在离散和连续模型中保持不变。

---

## 3. 拉曼系数（公式3.1.5）

```
g_R(f₁,f₂) = g_eff(f₁,f₂) / A_eff(f₁,f₂)
```

g_eff：拉曼增益效率，可从实验数据或拟合模型获取
A_eff：有效模场面积

---

## 4. 离散SpRS噪声模型

### 4.1 前向噪声（公式3.1.3）

单泵浦贡献（频率 f₂ 的泵浦在 f₁ 处产生的噪声）：

当 α₁ = α₂ 时：
```
P_{f,2→1}(z) = (P_{f,2} + P_{b,2}) × σ_{1,2} × exp(-α₁×z) × z
```

当 α₁ ≠ α₂ 时：
```
P_{f,2→1}(z) = (P_{f,2} + P_{b,2}) × σ_{1,2} × exp(-α₁×z) × [1 - exp(-(α₂-α₁)×z)] / (α₂ - α₁)
```

实现说明：α₁ = α₂ 的情况是 α₁ ≠ α₂ 公式在 α₂→α₁ 极限下的洛必达结果。代码中可统一使用 α₁ ≠ α₂ 的公式，当 |α₂-α₁| < 阈值时切换到 α₁ = α₂ 的公式，避免除零。

### 4.2 总前向噪声（公式3.1.6）

对所有泵浦信道求和：
```
P_{f,1}(z) = Σ_{f₂} P_{f,2→1}(z)
```

### 4.3 后向噪声（公式3.1.7）

```
P_{b,2→1}(z) = (P_{f,2} + P_{b,2}) × σ_{1,2} × [1 - exp(-(α₂+α₁)×z)] / (α₂ + α₁)
```

注意：后向公式中分母是 α₂+α₁（求和），而前向公式中是 α₂-α₁（求差）。

### 4.4 总后向噪声（公式3.1.8）

```
P_{b,1}(z) = Σ_{f₂} P_{b,2→1}(z)
```

---

## 5. 连续SpRS噪声模型

### 5.1 前向噪声PSD（公式3.2.1）

当 α₁ = α₂ 时：
```
G_{f,2→1}(z) = (G_{f,2} + G_{b,2}) × σ_{1,2} × exp(-α₁×z) × z
```

当 α₁ ≠ α₂ 时：
```
G_{f,2→1}(z) = (G_{f,2} + G_{b,2}) × σ_{1,2} × exp(-α₁×z) × [1 - exp(-(α₂-α₁)×z)] / (α₂ - α₁)
```

G_{f,2→1} [W/Hz]：频率 f₂ 的泵浦在频率 f₁ 处产生的SpRS噪声PSD
G_{f,2}, G_{b,2} [W/Hz]：泵浦光PSD

注意：连续模型与离散模型的公式结构完全相同，只是将功率 P 替换为 PSD G。σ_{1,2} 的定义不变。

### 5.2 总前向噪声功率（公式3.2.2）

对泵浦频率积分：
```
P_{f,1}(z) = ∫ G_{f,2→1}(z) df₂
```

### 5.3 后向噪声PSD（公式3.2.3）

```
G_{b,2→1}(z) = (G_{f,2} + G_{b,2}) × σ_{1,2} × [1 - exp(-(α₂+α₁)×z)] / (α₂ + α₁)
```

### 5.4 总后向噪声功率（公式3.2.4）

```
P_{b,1}(z) = ∫ G_{b,2→1}(z) df₂
```

---

## 6. 交叉验证关系

离散模型和连续模型的关系：
- 当信道带宽 B_s → 0 时，G_{f,2} × B_s → P_{f,2}
- 连续模型的积分退化为离散模型的求和
- 测试方法：用极窄矩形谱的连续模型对比等功率的离散模型

数值验证检查项：
- SpRS噪声功率应随泵浦功率线性增长（一次方关系）
- 斯托克斯方向（低频侧）噪声应大于反斯托克斯方向（高频侧）
- 噪声PSD量级通常在 10⁻⁸ ~ 10⁻¹² mW/GHz（取决于泵浦功率和频率间隔）
