# 建模假设与讨论

## 基本假设

1. **DWDM等间隔系统**：信道等间隔分布，间隔为50 GHz或100 GHz，符合商用光模块设计现状。

2. **无放大器**：发射端和接收端之间无光放大器（单跨段无放大链路）。

3. **无色散补偿**：不关注色散管理，无色散补偿光纤（uncompensated transmission）。

4. **参数波长依赖性**：考虑衰减α、折射率n和传播常数β的波长依赖性，不考虑它们随传播距离的变化。

5. **群速度假设**：假定不同频率的光具有相同的群速度（忽略walk-off效应）。

6. **量子信道功率可忽略**：量子信道功率远小于经典信道（≈真空态），不参与非线性过程的产生。量子信道不产生噪声贡献，但接收经典信道产生的FWM噪声和SpRS噪声。不考虑量子信道受到的受激拉曼散射(SRS)噪声。

## 量子噪声计算的特殊性

- **不存在SCI**：量子信道功率≈0，不产生自信道非线性干扰
- **只计算信道间效应**：FWM噪声来自经典信道频率分量的混频落入量子频率
- **SpRS来自经典泵浦**：所有经典信道作为泵浦源，在量子频率处产生散射噪声
- 本模型是经典GN-Model框架在 P_CUT → 0 极限下的自然简化

## 工作创新点

1. 考虑经典数据信号的**连续频谱特性**（非离散单频），得到噪声的频谱分布
2. 考虑经典数据信号的**频谱形状**（升余弦滚降、OSA实测），而非简单矩形谱
3. 基于噪声频谱分布提出量子信道**最优频率位置**的资源分配方案
4. 通过共纤传输实验验证

## 未来扩展方向

- 引入EGN修正：通过调制格式的四阶矩(Φ_a)和六阶矩(Ψ_a)修正FWM噪声预测
- 考虑SRS功率倾斜对FWM噪声的影响
- SSFM数值验证
